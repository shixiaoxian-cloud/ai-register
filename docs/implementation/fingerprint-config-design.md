# 浏览器环境配置设计说明

> 迁移说明：本文件保留旧文件名以兼容历史引用。当前正式设计以 `BrowserEnvironmentConfig` 为唯一真值。

## 设计目标

将浏览器环境配置定义为授权测试中的平台资源，要求：

- 字段可审计
- 配置可解释
- 来源可追踪
- 审批可校验
- 运行可回显

## 资源模型

核心字段：

- 资源标识与名称
- 来源类型与来源标签
- 审批状态、批准人、批准时间
- 浏览器名称、浏览器版本、平台
- `UA`
- `UA-CH`
- `locale`
- `timezone`
- `viewport`
- `screen`
- `geolocation`

## 绑定关系

- 每个 `Plan` 必须显式绑定一个浏览器环境配置
- 被方案引用的浏览器环境配置不能删除

## 运行前校验

执行前必须校验：

- 是否已绑定
- 是否已批准
- 白名单字段是否完整
- 字段间是否存在明显冲突

校验失败时直接阻断执行。

## 运行回显

运行中与产物中都要回显：

- 配置名称
- 浏览器版本
- 来源类型
- 审批状态
- 生效字段摘要

## 当前非目标

以下内容不属于当前设计：

- 深层浏览器伪装
- 外部诊断网站驱动验收
- 任意字段自由透传
- 多配置随机轮换
