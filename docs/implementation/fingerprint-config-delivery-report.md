## ⚠️ 注意事项

### 安全和合规

**仅用于：**
- ✅ 授权的安全测试
- ✅ 测试自己的应用
- ✅ 教育和研究目的
- ✅ 合法的自动化测试

**禁止用于：**
- ❌ 未经授权的系统访问
- ❌ 绕过付费墙或访问限制
- ❌ 大规模数据抓取
- ❌ 任何违反服务条款的行为

### 指纹一致性

**重要：** 确保指纹各部分一致

```
✅ 正确示例：
User Agent: Chrome 131 (Windows)
Platform: Win32
Timezone: America/Los_Angeles
IP: 美国洛杉矶

❌ 错误示例：
User Agent: Chrome 131 (Windows)
Platform: MacIntel  ← 不一致！
Timezone: Asia/Shanghai  ← 与 IP 不匹配！
IP: 美国洛杉矶
```

### 定期更新

浏览器版本更新快，建议：
- 每月检查一次内置模板
- 使用导出工具重新收集真实浏览器指纹
- 关注浏览器版本发布公告
