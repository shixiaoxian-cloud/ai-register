# Design Workspace

本目录承接 `ai-pm` 的设计与交付产物，默认不直接放生产代码。

- `design-tokens.md`：共享设计语言和视觉约定
- `screens/`：确认后的静态页面稿
- `handoff/`：给开发的交付说明
- `ui_ux/`：设计指导文档和待定项
- `prototypes/`：交互原型
- `ai_prompts/`：用于设计生成或模拟的提示词
- `tech_design/`：设计相关的技术设计稿

原则：

- 视觉和交互先在这里收敛，再决定是否进入 `src/`
- `openspec/design.md` 写的是本次 change 的技术设计，不替代这里的产品/体验设计资产
