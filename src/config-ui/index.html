运行说明

推荐的使用顺序如下，能避免因为漏配地址或环境变量导致的无效执行。

Runbook
  1. 先保存一个完整的 https:// 测试站点地址。
  2. 确认 `.env` 中已经填好测试账号、邮箱和 IMAP 配置。
  3. 选择运行模式并点击“开始测试”。
  4. 测试完成后点击“打开测试报告”查看详情。

实时日志

这里会滚动显示当前或最近一次测试任务的输出，便于快速定位卡在哪一步。

最多保留最近 400 条日志。测试运行中会自动刷新。
