## ADDED Requirements

### Requirement: 管理浏览器环境配置资源
系统 MUST 将浏览器环境配置作为一类独立平台资源进行管理，并在配置中心提供列表、详情、创建、编辑、删除、引用关系查看、来源展示、审批状态展示和更新时间展示。每条浏览器环境配置 MUST 记录来源类型、来源摘要、审批标记、版本信息、创建时间、更新时间和最近一次变更审计信息。

#### Scenario: 从本地真实浏览器导入环境配置
- **WHEN** 操作员提交来自本地真实浏览器导出的浏览器环境配置
- **THEN** 系统 MUST 创建一条浏览器环境配置记录，并保留来源类型、来源摘要、审批状态和审计信息

#### Scenario: 从组织批准模板创建环境配置
- **WHEN** 操作员选择组织批准模板创建浏览器环境配置
- **THEN** 系统 MUST 将模板标识、模板版本、审批标记和更新时间写入资源详情，并在配置中心可见

### Requirement: 浏览器环境配置只允许保存白名单字段
系统 MUST 只允许保存可审计、可解释且与测试环境一致的浏览器环境字段，包括 `UA`、`UA-CH`、`locale`、`timezone`、`viewport`、`screen`、`geolocation`、浏览器版本及其必要元数据。系统 MUST 拒绝 `stealthMode`、Canvas/WebGL/Audio 噪声注入、插件伪造、`navigator` 深度篡改、随机轮换及其他绕过导向字段。

#### Scenario: 提交包含禁止字段的环境配置
- **WHEN** 操作员提交包含 `stealthMode`、Canvas/WebGL/Audio 噪声或等价禁止字段的浏览器环境配置
- **THEN** 后端 MUST 以 `4xx` 状态阻断保存，并返回稳定的字段级错误信息

#### Scenario: 提交包含白名单外字段的环境配置
- **WHEN** 操作员提交未在浏览器环境白名单中的字段
- **THEN** 系统 MUST 拒绝该请求，而不是静默忽略、自动裁剪或以隐式默认值掩盖问题

### Requirement: 测试方案必须显式绑定浏览器环境配置
系统 MUST 支持 `Plan` 显式绑定浏览器环境配置，并在引用关系存在时阻止删除被引用的配置。配置中心和方案编辑界面 MUST 显示当前绑定关系与引用计数。

#### Scenario: 保存带浏览器环境配置绑定的方案
- **WHEN** 操作员为某个测试方案选择一条浏览器环境配置并保存
- **THEN** 系统 MUST 持久化该绑定关系，并在方案详情和浏览器环境配置详情中都可见

#### Scenario: 删除仍被方案引用的浏览器环境配置
- **WHEN** 操作员尝试删除仍被一个或多个测试方案引用的浏览器环境配置
- **THEN** 系统 MUST 拒绝删除，并返回被引用方案的可读提示信息
